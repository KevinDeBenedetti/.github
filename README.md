# .github

> Default community health files for all repositories under [@KevinDeBenedetti](https://github.com/KevinDeBenedetti).

## Features

- `CODE_OF_CONDUCT.md` — Contributor Covenant v2.1
- `CONTRIBUTING.md` — how to report bugs, suggest features, and submit pull requests
- `SECURITY.md` — supported versions, vulnerability reporting, and security best practices
- `SUPPORT.md` — where to get help (Discussions, issues, docs)
- Issue templates: 🐛 Bug report and ✨ Feature request (blank issues disabled)
- `PULL_REQUEST_TEMPLATE.md` — change description, type classification, and checklist
- Discussion templates: announcements, general, and Q&A
- `CODEOWNERS` — assigns `@KevinDeBenedetti` as default reviewer
- `FUNDING.yml` — links [GitHub Sponsors](https://github.com/sponsors/KevinDeBenedetti)

## Usage

GitHub automatically applies these files as defaults to any repository under this account that does not define its own version. A per-repo file always takes precedence.

> Reusable workflows and composite actions live in a separate repository: [`github-workflows`](https://github.com/KevinDeBenedetti/github-workflows).
